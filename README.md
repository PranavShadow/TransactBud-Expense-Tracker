# TransactBud - Expense Tracker

A modern, full-featured expense tracking application built with Next.js and modern web technologies. TransactBud helps you manage your finances effectively with an intuitive interface and powerful features.

## Tech Stack

### Frontend
- **Framework**: Next.js 16.2.6
- **React**: 19.2.4
- **Styling**: Tailwind CSS 4, CSS
- **UI Components**: shadcn, Radix UI
- **Icons**: Lucide React

### Backend & Database
- **ORM**: Drizzle ORM 0.45.2
- **Database**: PostgreSQL (Neon Serverless)
- **Authentication**: Clerk

### Additional Libraries
- **Charts**: Recharts 3.8.1
- **Theme Management**: next-themes 0.4.6
- **Toast Notifications**: Sonner 2.0.7
- **Emoji Picker**: emoji-picker-react 4.19.1
- **Utilities**: clsx, class-variance-authority, tailwind-merge


## Installation

**Clone the repository**
```bash
git clone https://github.com/PranavShadow/TransactBud-Expense-Tracker.git
cd TransactBud-Expense-Tracker
```
